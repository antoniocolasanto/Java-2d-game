🎮 2D Platformer Game (Java + MongoDB)

Un videogioco platform 2D sviluppato in Java, ispirato allo stile classico di Super Mario, con gestione dei dati tramite MongoDB in locale.

🚀 Descrizione

Questo progetto è un gioco 2D in cui il giocatore controlla un personaggio attraverso livelli ricchi di ostacoli, nemici e piattaforme. Il gameplay richiama i classici platform arcade, con meccaniche di salto, movimento laterale e raccolta di oggetti.

A differenza dei classici giochi standalone, il progetto integra un database MongoDB locale per salvare e gestire informazioni come punteggi, progressi del giocatore e dati di gioco.

🧩 Caratteristiche principali
🎮 Gameplay stile platform 2D (movimento, salto, collisioni)
🧱 Sistema di livelli con piattaforme e ostacoli
👾 Nemici con comportamenti base
⭐ Raccolta di oggetti e punteggio
💾 Salvataggio dati tramite MongoDB locale
🧑‍💻 Sviluppato interamente in Java (OOP)
🛠️ Tecnologie utilizzate
Java – logica di gioco e rendering
Java Swing
MongoDB – database NoSQL locale
MongoDB Java Driver – connessione al database
🗄️ Utilizzo del Database

Il database MongoDB viene utilizzato per:

Salvare i punteggi dei giocatori
Memorizzare i progressi di gioco
Gestire eventuali impostazioni o statistiche per classifica
